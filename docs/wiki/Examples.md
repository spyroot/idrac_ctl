# Examples

Author: Mus <spyroot@gmail.com>

The `examples/` directory is a recipe book. Read a script before running it, because several examples
stage BIOS, boot, storage, virtual media, or power changes.

## Read-First Scripts

- `examples/example_discover_host.sh` walks a new host.
- `examples/example_inventory_walk.sh` collects system, chassis, PCI, storage, drive, and boot state.
- `examples/example_sensors_read.sh` reads sensors and filters common hardware readings.
- `examples/example_console_access.sh` shows console metadata.
- `examples/hpe_ilo_canary.sh` runs the HPE emulator canary with read-only checks and a dry-run reset.
